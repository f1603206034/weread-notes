# 第14章 竞争市场上的企业

> 📌 6.纽约的椒盐卷饼摊是实现了长期均衡的完全竞争行业。有一天，市政府开始对每个摊位每月征收100美元的税。这种政策在短期和长期中会如何影响椒盐卷饼的消费量？（　）
a.短期中减少，长期中没有变化
b.短期中增加，长期中没有变化
c.短期中没有变化，长期中减少
d.短期中没有变化，长期中增加
【答案】c
> ⏱ 2025-11-18 10:13:16

💭 征税后的成本变化情况
◦ 固定成本有所增加，不过平均可变成本（AVC）曲线和边际成本（MC）曲线并未改变。
◦ 平均总成本（ATC）曲线会向上移动。
◦ 短时间内，市场价格  P  高于平均可变成本（AVC）时，企业不会对自身的生产决策做出调整。
◦ 又由于市场价格  P  低于新的平均总成本（ATC），企业会选择退出市场。
这一行为会使得市场的供给曲线向左移动，进而导致市场价格上升，销量下降。
市场稳定后的状态
◦ 单个企业最终会在原均衡点的右上方达成新的均衡，此时价格上涨，但销量也会上升。
那么为什么，企业和市场的新的均衡点会出现这种矛盾的移动现象？
解答：
让我为您澄清这个看似矛盾的地方。您完全正确地指出了两点：
对于单个企业来说：新的长期均衡点确实是在新的ATC曲线的最低点，这个点位于原均衡点的右上方。
对于整个市场来说：新的长期均衡点位于原均衡点的左上方（价格更高，总产量更低）。
那么，这两个看起来矛盾的结论是如何统一的呢？
关键在于区分 单个企业的产量 (q) 和 市场的总产量 (Q)。
我们来一步步分解这个过程：
阶段一：初始长期均衡
市场：市场供给曲线S1和需求曲线D相交，决定了市场价格P1和市场总产量Q1。
单个企业：企业是价格接受者，面对价格P1。此时，P1 = MC = ATC1的最低点。企业生产q1，经济利润为零。
关系：Q1 = n1 * q1 (其中 n1 是市场中的企业数量)
阶段二：征收100美元固定税（短期）
市场：没有变化。价格仍是P1，总产量仍是Q1。
单个企业：
MC曲线不变。
ATC曲线上移至ATC2。
由于价格P1现在低于ATC2，企业开始亏损。
阶段三：向新的长期均衡调整
这是解开谜团的关键一步。
企业退出：由于行业内所有企业都在亏损，最无法承受的企业开始退出市场。
市场供给曲线移动：随着企业数量的减少，市场总供给曲线S1开始向左移动。假设最终移动到S2。
市场价格上涨：市场供给减少导致了“物以稀为贵”，市场价格开始从P1上涨。
何时停止退出？：企业会一直退出，市场价格会一直上涨，直到剩下的企业不再亏损为止。这个点就是当市场价格正好等于新的ATC曲线（ATC2）的最低点时。我们称这个新价格为P2。
阶段四：新的长期均衡
市场：
新的均衡点是S2和D的交点。
价格为 P2 (P2 > P1)。
市场总产量为 Q2 (Q2 < Q1)。
这解释了为什么市场的均衡点移动到了“左上方”。
单个（幸存的）企业：
它们现在面对的是由市场决定的新价格P2。
在这个价格水平上，P2 = MC = ATC2的最低点。
为了达到这个利润最大化（此时是零利润）的点，每个幸存企业会调整自己的产量到q2。
观察图表可以发现，ATC2的最低点 (q2) 确实在ATC1的最低点 (q1) 的右边。所以，q2 > q1。
这解释了为什么单个企业的均衡点移动到了“右上方”。
核心解惑：如何调和“右上”和“左上”？
这里的关键在于企业数量 n 的急剧减少。
初始状态：市场总产量 Q1 = n1 * q1
最终状态：市场总产量 Q2 = n2 * q2
虽然每个幸存下来的企业产量增加了 (q2 > q1)，但是企业的数量大幅减少了 (n2 < n1)。
企业数量减少的影响，远远超过了单个企业产量增加的影响，最终导致市场的总产量是下降的。
举个例子：
初始：市场有 100 个摊位 (n1)，每个卖 50 个卷饼 (q1)。市场总产量 Q1 = 100 * 50 = 5000。
征税后：一部分摊位退出，市场上只剩下 60 个摊位 (n2)。为了弥补成本，价格上涨，每个摊位现在必须卖 70 个卷饼 (q2) 才能保本。
最终：市场总产量 Q2 = 60 * 70 = 4200。
结论：
单个企业：为了在更高的成本（ATC2）下生存，必须等到市场价格上涨，并扩大生产到新的、效率更高的规模点（ATC2的最低点），所以它的产量 q 增加了。
整个市场：由于大量企业退出了，即使剩下的企业生产得更多，市场的总产量 Q 还是减少了。

# 第15章 垄断

> 📌 iii.对每张CD支付150Ectenia元，因而垄断者的边际成本变为MC=100+10Q+150Q=100+160Q，由MC=MR可得：100+160Q=1000-20Q，从而可以解得Q=5，P=950Ectenia元。此时，产量更低，价格更高，因而无谓损失变大了。
> ⏱ 2025-11-18 20:26:45

💭 答案错误，正确解法如下：
1. 边际成本（MC）调整原理
原总成本（TC原）：
TC原 = FC + VC = FC + (100Q + 5Q²)
边际成本（MC原）：
MC原 = d(TC原)/dQ = 100 + 10Q
新增成本：每售出1张CD支付150元 → 新增可变成本：
ΔTC = 150Q
新总成本（TC新）：
TC新 = FC + (100Q + 5Q²) + 150Q = FC + 250Q + 5Q²
新边际成本（MC新）：
MC新 = d(TC新)/dQ = 250 + 10Q
2. 利润最大化计算
需求函数：
P = 1000 - 10Q
边际收益（MR）：
MR = d(TR)/dQ = 1000 - 20Q
均衡条件：
MR = MC新 → 1000 - 20Q = 250 + 10Q
解得：
30Q = 750 → Q = 25张
P = 1000 - 10×25 = 750元
3. 社会最优产量（Q有效）
条件：需求 = 新边际成本
1000 - 10Q = 250 + 10Q → 20Q = 750 → Q有效 = 37.5张
4. 无谓损失（DWL）计算
公式：
DWL = 0.5 × (P垄断 - MC垄断) × (Q有效 - Q垄断)
代入数值：
P垄断 = 750元
MC垄断 = 250 + 10×25 = 500元
Q有效 = 37.5张
Q垄断 = 25张
计算：
DWL = 0.5 × (750-500) × (37.5-25) = 0.5 × 250 × 12.5 = 1562.5元
