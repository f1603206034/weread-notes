本书是按照教育部大学数学课程教学指导委员会的基本要求，充分吸取当前高等数学教材的精华，并结合同济大学数学系多年来的教学实践经验，针对当前学生的知识结构和习惯特点而编写的。全书分为上、下两册。本书为下册，是多元函数微积分部分，共四章，主要内容包括向量与空间解析几何，多元函数微分学，多元函数积分学，无穷级数。每节前面配有课前导读，核心知识点配备微课，每章后面附有章节测试和拓展阅读。本书注重知识点的引入方法，使之符合认知规律，更易于读者接受。同时，本书精炼了主要内容，对部分内容调整了顺序，使结构更加简洁，思路更加清晰。本书还注重知识的连贯性，例题的多样性和习题的丰富性、层次性，使读者在学习数学知识点的同时拓宽视野，欣赏数学之美。本书可作为高等院校理工科类各专业的教材，也可作为社会从业人员的自学参考用书。

# 第三节 复合求导、隐函数求导及方向导数

> 📌 梯度
> ⏱ 2025-05-22 22:02:56

💭 tī dù
对于函数f（x，y，z），向量场称为f（x，y，z）的梯度，记作gradf或▽f。它的方向是f的变化率（方向导数）最大的方向，它的长度表示f的最大变化率。例如，f表示电势，则－gradf表示电场强度。
关键点：
梯度是一个向量。
梯度的正方向是函数在该点增长最快的方向。梯度的反方向是函数在该点下降最快的方向。
梯度的大小（模）是函数在该方向上的增长速率（坡度）。
想象你在一个山坡上，你想知道哪个方向是最陡峭的向上爬的路。梯度就能告诉你这个方向，并且告诉你这个方向有多陡。
在数学上，对于一个多变量函数（比如 f(x, y)，它的值会随着 x 和 y 的变化而变化），梯度是一个向量（vector），它指向函数值增长最快的方向，并且这个向量的长度（或称为模）代表了在这个方向上函数值增长的速率（即坡度有多陡）。
一个简单的数学例子：
假设我们有一个二维函数 f(x, y) = x² + y²。
这个函数可以想象成一个三维空间中的碗状曲面，碗底在原点 (0,0)。f(x,y) 的值代表了在点 (x,y) 处的高度。
计算偏导数：
为了找到梯度，我们首先需要计算函数 f 对每个变量的偏导数：
∂f/∂x (f对x的偏导数)：表示当y保持不变时，f随x变化的速率。
∂f/∂x = d(x² + y²)/dx = 2x (把y²看作常数)
∂f/∂y (f对y的偏导数)：表示当x保持不变时，f随y变化的速率。
∂f/∂y = d(x² + y²)/dy = 2y (把x²看作常数)
梯度的表示：
函数 f(x, y) 的梯度，通常用 ∇f (读作 "nabla f" 或 "del f") 表示，是一个由这些偏导数组成的向量：
∇f(x, y) = [∂f/∂x, ∂f/∂y] = [2x, 2y]
在特定点计算梯度：
让我们选一个具体的点，比如说点 P(1, 2)。
在这个点，函数的值是 f(1, 2) = 1² + 2² = 1 + 4 = 5。
这个点上的梯度是：
∇f(1, 2) = [2*1, 2*2] = [2, 4]
解释梯度向量 [2, 4] 的含义：
方向 (Direction)： 向量 [2, 4] 指向从点 (1, 2) 出发，使函数 f(x, y) 值增长最快的方向。如果我们在 xy 平面上的点 (1, 2) 处，沿着向量 [2, 4] 代表的方向（即x方向移动2个单位，y方向移动4个单位的那个方向）移动一小步，函数值 f（即高度）的增加会是最大的。对于 f(x,y) = x^2 + y^2 这个碗状曲面，这个方向就是从 (1,2) 直接指向远离原点的方向。
大小/模 (Magnitude)： 向量 [2, 4] 的大小（或模）是 ||∇f(1, 2)|| = √(2² + 4²) = √(4 + 16) = √20 ≈ 4.47。
这个值 √20 表示在点 (1, 2) 处，沿着梯度方向（即最陡峭的上升方向）移动时，函数值 f 的增长速率。也就是说，如果你在最陡峭的方向上移动一个极小的单位距离，函数值 f 大约会增加 √20。
梯度的实际意义：
梯度在许多领域都有非常重要的实际意义：
优化问题 (Optimization)：
梯度下降法 (Gradient Descent)： 这是机器学习和深度学习中最核心的算法之一。当我们要最小化一个损失函数（cost function）时，我们会计算损失函数相对于模型参数的梯度。然后，我们沿着梯度的相反方向（因为梯度指向增长最快的方向，相反方向就是下降最快的方向）更新参数，从而逐步找到损失函数的最小值。想象你在山谷里，想走到谷底，每一步你都会选择当前位置最陡峭的下坡方向走。
梯度上升法 (Gradient Ascent)： 如果目标是最大化一个函数（比如奖励函数），则会沿着梯度的方向更新参数。
物理学：
热传导： 热量总是从温度高的地方流向温度低的地方。温度的梯度指向温度增长最快的方向，所以热流的方向与温度梯度的方向相反。
电场与电势： 电场强度是电势的负梯度。电场线指向电势下降最快的方向。该点的梯度的模就是电场强度的大小。
流体力学： 压力梯度驱动流体的流动。
图像处理：
边缘检测： 图像中的边缘通常是像素强度发生剧烈变化的地方。计算图像强度函数的梯度，梯度值大的地方就可能对应着图像的边缘。
地理信息系统 (GIS) 和地形分析：
坡度与坡向： 地形表面某一点的梯度可以给出该点的坡度（梯度的大小）和坡向（梯度的方向），这对于分析水流方向、滑坡风险等非常重要。
寻找函数的极值点：
在单变量微积分中，我们通过令导数等于零来找极值点。在多变量微积分中，函数在某个点取得极值（最大值或最小值）的必要条件是该点的梯度为零向量（即所有偏导数都为零）。这样的点称为驻点或临界点。
